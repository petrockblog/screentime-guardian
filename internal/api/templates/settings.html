Configuration File

The daemon reads configuration from /etc/screentime-guardian/config.yaml

Example Configuration 
listen_addr: ":8080"
database_path: "/var/lib/screentime-guardian/data.db"
admin_password: "your-secure-password"
warning_intervals:
  - 5
  - 1
check_interval: 30s
grace_period: 1m
mDNS Discovery

This interface can be accessed at:

  • http://screentime-guardian.local:8080 (when mDNS is enabled)
  • Or use the server's IP address directly
Requires avahi-daemon on the server and Bonjour/mDNS on client devices.
